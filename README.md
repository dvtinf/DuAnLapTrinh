# 🚀 C++ & Multi-language Online Judge System

Hệ thống chấm bài lập trình trực tuyến mạnh mẽ, an toàn và hiệu suất cao.

## ✨ Tính năng nổi bật
- **⚡ Tốc độ cực nhanh:** Tải trang < 2s, chấm bài thời gian thực.
- **🛡️ Bảo mật tuyệt đối:** Sandbox Docker cô lập hoàn toàn mã nguồn người dùng.
- **📊 Quản lý toàn diện:** Dashboard cho Học sinh, Giáo viên và Admin.
- **🤖 AI Ready:** Hạ tầng sẵn sàng tích hợp AI Code Assistant.
- **🌐 Đa ngôn ngữ:** Hỗ trợ C++17, Python 3, Java 17.

## 🛠️ Công nghệ sử dụng
- **Frontend:** Vanilla JS (ES2023), Bootstrap 5, Monaco Editor.
- **Backend:** Firebase Cloud Functions, Firestore.
- **CI/CD:** GitHub Actions.
- **Judge Core:** Docker, GCC.
