# 2. Development Philosophy


Dự án được phát triển theo nguyên tắc:


```
Build Small

        |

Validate

        |

Improve

        |

Expand

```


Ưu tiên:

1. Hoàn thiện chức năng chính.
2. Đảm bảo tính ổn định.
3. Tối ưu hiệu năng.
4. Mở rộng tính năng nâng cao.
