# HƯỚNG DẪN DÀNH CHO QUẢN TRỊ VIÊN

## 1. Quản lý người dùng [21]
- Thay đổi quyền hạn (Role) của người dùng từ Student lên Teacher.
- Khóa (Lock) các tài khoản vi phạm quy định.

## 2. Giám sát hệ thống Judge [1]
- Theo dõi hàng đợi chấm bài (Judge Queue) để đảm bảo không bị nghẽn.
- Kiểm tra nhật ký hoạt động (Audit Logs) để truy vết các thao tác quan trọng.

## 3. Cấu hình hệ thống
- Cập nhật thông báo chung toàn website tại `/admin/announcements`.
- Quản lý các tham số cấu hình trong tệp `settings` trên Firebase.
