# HƯỚNG DẪN DÀNH CHO GIÁO VIÊN

## 1. Quản lý bài tập [21]
- Truy cập `/admin/problems` để tạo bài tập mới.
- Nhập tiêu đề, nội dung đề bài (hỗ trợ định dạng Markdown).
- Thiết lập độ khó (Dễ, Trung bình, Khó) và các giới hạn kỹ thuật.

## 2. Thiết lập Test Case [57]
- Với mỗi bài tập, giáo viên cần tải lên bộ Test Case (Input/Output).
- Có thể thiết lập Test Case **Ẩn** hoặc **Công khai**.
- Thiết lập điểm số tương ứng cho từng Test Case.

## 3. Giám sát học sinh
- Xem thống kê tỷ lệ làm bài đúng của lớp.
- Sử dụng chức năng **"Chấm lại bài" (Rejudge)** nếu cần thay đổi bộ Test Case.
